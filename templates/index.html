How It Works

1. Ask Claude

"Can you analyze my health records?"

2. Connect Your Records

Claude provides a secure link. Click it, select your provider, sign in.

3. Get Insights

  • Current medications and what they're for
  • Lab result trends over time
  • Conditions explained in plain language
  • Preventive care that might be due
  • Clinical notes from your visits

Architecture

User asks Claude ‚Üí Claude creates session ‚Üí User clicks link ‚Üì User signs into patient portal (Epic MyChart, etc.) ‚Üì SMART on FHIR fetches health data ‚Üì Data sent to this server, Claude polls until ready ‚Üì Claude analyzes: labs, meds, conditions, notes

For Developers

git clone https://github.com/jmandel/health-skillz
cd health-skillz
cp config.json.example config.json
# Edit config.json with your client IDs
bun install
bun run setup
bun run start

API

  • POST /api/session - Create session
  • GET /api/poll/{id} - Poll for data
  • POST /api/data/{id} - Receive data
